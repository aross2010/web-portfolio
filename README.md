## How to Use

1. Set up your own [Resend](https://react.email/docs/integrations/resend) API Key as RESENED_API_KEY in a .env file. 
2. Add your own EMAIL in a .env file.
3. Customize to your liking — insert your own data in data.ts, edit the content in the inner elements, and select a primary color in tailwind.config.ts.
